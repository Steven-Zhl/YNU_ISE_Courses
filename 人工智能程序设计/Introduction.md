# 人工智能程序设计

> 本课其实就是Python程序设计，是一并未涉及到什么人工智能的知识
>
> Python是一门功能丰富的编程语言，目前我主要使用的编程语言就是Python，虽然最近越发感到它的局限性，但仍然瑕不掩瑜，它是一门优秀且易用的编程语言

## 使用环境

* Windows 10
* Python 3.9.10 (Based on Anaconda3)
  * openpyxl
  * PIL
  * tkinter
  * opencv-python
